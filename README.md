# NYAKWIT 

Frontend aplikasi deteksi penyakit daun sawit menggunakan React + Vite + TailwindCSS.

## Features
- Upload gambar daun sawit
- Deteksi penyakit AI
- Dashboard admin
- Responsive UI

## Tech Stack
- React
- Vite
- TailwindCSS
- TypeScript
- Supabase

## Installation
npm install
npm run dev
